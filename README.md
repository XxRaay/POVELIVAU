# ПОВЕЛЕВАЮ 📜

> Язык программирования на синтаксисе древнерусского слога

```
Свиток Сказа "Приветствие"

Глаголю народу: "Слава Руси! Сие есть первая программа на языке Повелеваю!",

Повелеваю: Начать выполнение!
```

---

## Запуск

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить программу
python main.py examples/hello.pov

# С отладкой (выводит токены и AST)
python main.py program.pov --debug

# Интерактивный REPL
python main.py --repl

# Запустить тесты
python tests/test_interpreter.py

# Запустить веб-IDE (онлайн редактор)
python server.py
# Браузер откроется автоматически, либо зайдите на:
#   http://localhost:8000
```

**Требования:** Python 3.10+, библиотека `requests`

## Полный синтаксис языка

### Структура программы

```
Свиток <Тип> "<Название>"

... тело программы ...

Повелеваю: Начать выполнение!
```

### Правила

- Каждая инструкция заканчивается запятой `,`
- Блоки (`if`, `while`, функции) заканчиваются двоеточием `:` и имеют отступ
- Отступы: 5 пробелов или 1 табуляция
- Имена переменных — с заглавной буквы: `Счёт`, `Имя`, `Загадка`
- Комментарии: `// текст`
- Строки в двойных кавычках; интерполяция: `"Привет, [Имя]!"`

---

### Присвоение

```
// Создать переменную
Повелеваю: Отныне 42 именоваться Ответ,
Повелеваю: Отныне "Иван" именоваться Имя,

// Альтернативный синтаксис
Указываю: Сотворить сундук пустой, именем Счёт, и положить в него 0,
```

### Вывод

```
Глаголю народу: "Текст для вывода",
Вещаю: "Привет, [Имя]!",
Кричу на всю Русь: "Важное сообщение!",
```

Все три команды эквивалентны `print(...)`.

### Ввод

```
// Строка
Повелеваю: Взять слово от гостя заезжего и наречь Имя,

// Целое число
Повелеваю: Взять слово от гостя заезжего, оборотить его в Число Цельное и наречь Число,

// Дробное число
Повелеваю: Взять слово от гостя заезжего, оборотить его в Число Дробное и наречь Дробь,

// С подсказкой
Вопрошаю гостя: "Как тебя зовут? " и наречь Имя,
```

### Условия

```
Суд вершу:
     Коль Возраст меньше 18:
          Вещаю: "Юн ещё!",
     Иначе коль Возраст меньше 65:
          Вещаю: "В самом расцвете!",
     Иначе:
          Вещаю: "Умудрённый годами!",
```

**Операторы сравнения:**

| Язык | Python |
|------|--------|
| `в точности как` | `==` |
| `не ровня` | `!=` |
| `меньше` | `<` |
| `больше` | `>` |
| `не меньше` | `>=` |
| `не больше` | `<=` |

**Логические операторы:**

| Язык | Python |
|------|--------|
| `и к тому же` | `and` |
| `либо же` | `or` |
| `не есть` | `not` |

### Циклы

```
// Бесконечный цикл
Коловрат:
     Вещаю: "Навеки!",
     Повелеваю: Пресечь бег сего коловрата,

// While с условием
Коловрат покуда Счёт меньше 10:
     Добавить в сундук Счёт единицу малую,

// For по диапазону (включительно с обоих концов)
Странствие И по землям от 1 до 10:
     Вещаю: "[И]",

// For по списку
Странствие Элемент по дружине МойСписок:
     Вещаю: "[Элемент]",

// Управление циклом
Повелеваю: Пресечь бег сего коловрата,    // break
Повелеваю: Перейти к следующему витку,     // continue
```

### Функции

```
Умение нарекаю Приветствовать(Имя):
     Вещаю: "Здравствуй, [Имя]!",

Умение нарекаю Сложить(А, Б):
     Повелеваю: Вернуть А сложить с Б,

// Вызов
Призвать умение Приветствовать(Имя),

// Вызов с сохранением результата
Повелеваю: Отныне Призвать умение Сложить(10, 32) именоваться Итог,
```

### Списки (Дружины)

```
// Создание
Дружина именем Числа с ратниками [1, 2, 3, 4, 5],

// Добавление элемента
Принять в дружину Числа ратника 6,

// Удаление по индексу
Изгнать из дружины Числа ратника с позиции 0,

// Доступ по индексу
Повелеваю: Отныне Ратник под номером 0 в дружине Числа именоваться Первый,

// Длина
Повелеваю: Отныне длина дружины Числа именоваться Размер,
```

### Математика

```
Повелеваю: Отныне А сложить с Б именоваться Сумма,
Повелеваю: Отныне А отнять Б именоваться Разность,
Повелеваю: Отныне А умножить на Б именоваться Произведение,
Повелеваю: Отныне А разделить на Б именоваться Частное,
Повелеваю: Отныне А остаток от деления на Б именоваться Остаток,

// Инкремент / декремент
Добавить в сундук Счёт единицу малую,    // Счёт += 1
Убавить из сундука Счёт единицу,         // Счёт -= 1
Добавить в сундук Счёт 5,               // Счёт += 5
```

### Стандартная библиотека

```
// Случайное число [A, B]
Повелеваю: Отныне из палат случайных призвать число от 1 до 100 именоваться Брошенная,

// Преобразование типов
Повелеваю: Отныне оборотить Текст в Число Цельное именоваться Число,
Повелеваю: Отныне оборотить Число в Число Дробное именоваться Дробь,
Повелеваю: Отныне оборотить Число в Слово именоваться Текст,

// Длина
Повелеваю: Отныне длина дружины МойСписок именоваться Размер,

// Завершить программу
Повелеваю: Остановить всё и почить,
```

### Великие библиотеки (модули)

```text
// Подключить библиотеку по имени
Повелеваю: достать из библиотеки великой "http request",
Повелеваю: достать из библиотеки великой "земли",

// Подключить свиток (.pov) по названию из шапки
Повелеваю: достать из библиотеки великой свиток "Математический свиток",
```

Свитки ищутся в текущей земле исполнения и рядом с запускаемым свитком.

### Управление землями (файловая система)

```rust
// Листинг владений и путь
Повелеваю: Узнать имя земли текущей и наречь Путь,          // os.getcwd()
Повелеваю: Обозреть всё в пределе текущем и наречь Список,  // os.listdir()

// Смена земли
Повелеваю: Сменить землю на "C:/Заветы" и обосноваться там, // os.chdir("C:/Заветы")

// Возведение и разрушение чертогов и свитков
Повелеваю: Возвести чертог новый именем "Архив",            // os.mkdir("Архив")
Повелеваю: Предать забвению чертог "Старьё" со всеми свитками, // shutil.rmtree("Старьё")
Повелеваю: Изгнать из земель свиток "донос.txt",            // os.remove("донос.txt")
Повелеваю: Переименовать грамоту "черновик.txt" в "Указ_Финальный.txt", // os.rename(...)

// Вписать строку в txt‑свиток (append)
Повелеваю: Вписать в свиток "летопись_земель.txt" строку "Посещение земель завершено", // open(..., "a")

// Проверка границ владений
Суд вершу:
     Коль в землях есть "сокровище.db":                     // os.path.exists("сокровище.db")
          Вещаю: "Богатства на месте!",
     Иначе:
          Кричу на всю Русь: "Ограбили!",
```

### HTTP / Сеть

```
// GET-запрос: получить текст ответа по адресу
Повелеваю: Отныне из земель дальних призвать весть по адресу "https://example.com" именоваться Ответ,
Вещаю: "Получено: [Ответ]",

// POST-запрос: отправить весть с телом и получить текст ответа
Повелеваю: Отныне из земель дальних послать весть по адресу "https://example.com/api" с содержимым "Привет от Повелеваю" именоваться Ответ,
Вещаю: "Ответ: [Ответ]",
```

### Булевы значения

```
Повелеваю: Отныне Зрю истину именоваться Флаг,    // True
Повелеваю: Отныне Зрю ложь именоваться Флаг,      // False
Повелеваю: Отныне ничто именоваться Пусто,         // None
```

## Структура проекта

```
poveleyayu/
├── main.py          # Точка входа; REPL; --debug флаг
├── lexer.py         # Лексер: исходный текст → токены
├── parser.py        # Парсер: токены → AST (рекурсивный нисходящий)
├── interpreter.py   # Интерпретатор: паттерн Visitor
├── nodes.py         # Классы AST-узлов (dataclasses)
├── stdlib.py        # Стандартная библиотека
├── pov_modules/     # Реестр и модули «великих библиотек»
│   ├── http_request.py          # HTTP-модуль
│   └── zemli.py                 # Модуль управления землями
├── errors.py        # Древнерусские ошибки
├── examples/
│   ├── hello.pov                 # Hello World
│   ├── kalkulator.pov            # Калькулятор
│   ├── ugadai_chislo.pov         # Угадай число
│   ├── drujina.pov               # Функции и списки
│   ├── vesti_iz_zemel.pov        # HTTP / requests пример
│   └── ustav_upravleniya_zemlyami.pov # Устав Управления Землями (файловая система)
└── tests/
    └── test_interpreter.py  # Юнит-тесты
```
